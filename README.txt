Для запуска потребуются библиотеки sqlite3, PyQt5, pudub, librosa и pyqt5_tools.

На устройстве в папке с приложением должна находиться база данных "audio_1.db" и файл с графическим дизайном "project.ui".

Чтобы запустить программу, откройте файл "audio_editor.exe", база данных и графический дизайн так же должен находиться в локальном месте с файлом exe.

!!!Самое важное, перед работой нужно установить среду ffmpeg для обработки аудио, как это сделать написано в отдельном файле guid_for_ffmpeg!!!
